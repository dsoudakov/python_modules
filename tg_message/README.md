# Send message to Telegram Bot

## Use with:
```
pip install git+https://github.com/dsoudakov/python_modules.git@main#subdirectory=tg_message
in cli: tg_message "message text" <--debug>
--debug will print additional info on server response
In Telegram use BotFather and userinfobot (forward a message from your conversation with the new bot to this) to get bot token and chat_id
```
